== Description ==

People misspell their email address on a daily basis. Chances with be lower when using MailCheck for WordPress. This plugin will give a suggestion when a email address is being misspelled.

For example, when someone type john@doe.**con** it will give the suggestion for john@doe.**com**.


## Compatibility

**This plugin works by default with**

* WordPress registration
* WooCommerce registration
* WooCommerce checkout

## Translations

Translations are very welcome, best thing is, you only need to translate one string!


== Installation ==

## Manual installation

1. Upload the folder `mailcheck-for-wordpress` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the settings page to fine-tune the settings if desired

## Plugins page installation

1. Go to your site, login and go to the `plugins` page.
2. Click on `Add new`
3. Search for `MailCheck for WordPress`
4. Click on `Install now`
